# Durability

A browser-based educational game about reinforced concrete and the threats that degrade it — carbonation, chlorides, sulfates, and freeze-thaw cycles.

## About

You defend a reinforced concrete wall from falling contaminants by deflecting them with a paddle. Before each game you spec your concrete — cover depth, curing, mix, air content — within a budget, then survive as long as you can. Score enough points to unlock a protective Coating (100 pts) and Patch Repair (200 pts).

Built for fun by a civil engineer as a light-hearted way to show what makes good concrete last, and what makes bad concrete fail.

## Tech

A single self-contained HTML file — vanilla JavaScript, CSS, and SVG, no dependencies or build step. Runs in any modern browser.
